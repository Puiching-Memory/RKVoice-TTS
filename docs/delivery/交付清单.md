# 交付清单

## 1. 代码与脚本

- ASR 命令入口：uv run python -m scripts.delivery asr
- TTS 命令入口：uv run python -m scripts.delivery tts
- 板卡工具：uv run python scripts/board/set_board_static_ipv4.py
- 实际实现脚本：scripts/delivery、scripts/board、scripts/release
- 发布脚本：scripts/release/package_release.py
- Python 依赖定义：pyproject.toml、uv.lock

## 2. 文档

- 总览文档：README.md
- 指标文档：docs/requirements/项目指标.md
- 指标差距：docs/requirements/项目指标差距清单.md
- 选型报告：docs/reports/免费商用离线TTS技术选型详细汇报报告.md
- 交付文档：docs/delivery/部署手册.md、docs/delivery/验收手册.md、docs/delivery/回退手册.md
- 发布文档：docs/delivery/发布手册.md、docs/delivery/发布说明模板.md

## 3. 配置

- 示例配置：config/examples/board.local.env.example、config/examples/delivery.local.env.example
- 本地配置：config/local/

交付原则：

- config/local/ 不进入正式交付包。
- 真实密码、真实 IP、真实网卡名称等敏感参数不进入文档和源码默认值。

## 4. 产物

- 下载缓存：artifacts/cache/
- 离线源码包：artifacts/source-bundles/
- Unified runtime：artifacts/runtime/rkvoice_runtime/
- ASR 运行包：artifacts/runtime/rkvoice_runtime/asr/
- TTS 运行包：artifacts/runtime/rkvoice_runtime/tts/
- 回传结果：artifacts/runtime/rkvoice_runtime/asr/output/、artifacts/runtime/rkvoice_runtime/tts/output/

正式交付包建议只包含：

1. scripts/
2. docs/
3. config/examples/
4. README.md
5. pyproject.toml
6. uv.lock
7. RELEASE_NOTES.md

按需附带：

1. 已验证的运行包压缩包
2. 已验证的冒烟日志与音频样例
3. 已验证的模型版本说明

## 5. 不应直接交付的内容

- .venv/
- config/local/
- 构建缓存目录
- 临时调试脚本
- 含真实口令的任何文件

## 6. 交付前检查

1. 确认 scripts/delivery 和 scripts/board 下脚本可运行。
2. 确认 config/examples/ 中模板完整。
3. 确认 docs/delivery/ 文档与当前目录结构一致。
4. 确认 output 目录中的日志与音频可作为验收证据。
5. 确认本地敏感信息未混入正式文档或默认参数。
6. 如需对外发包，确认 scripts/release/package_release.py 已完成一次实际打包验证。
