# 验收手册

## 1. 验收输入

- 项目指标：docs/requirements/项目指标.md
- 指标差距：docs/requirements/项目指标差距清单.md
- 当前运行包：artifacts/runtime/paddlespeech_tts_armlinux_runtime/

## 2. 基础验收项

1. 能在开发 PC 上完成 download、build、upload 基本流程。
2. 能在 RK3588 板端生成音频文件。
3. 能将冒烟音频和测试日志自动回传到 PC。

## 3. 验收证据位置

- 冒烟日志：artifacts/runtime/paddlespeech_tts_armlinux_runtime/output/smoke_test.log
- 冒烟音频：artifacts/runtime/paddlespeech_tts_armlinux_runtime/output/smoke_test.wav

## 4. 当前推荐验收步骤

1. 执行 uv run python scripts/delivery/prepare_paddlespeech_tts_armlinux.py build
2. 执行 uv run python scripts/delivery/prepare_paddlespeech_tts_armlinux.py upload
3. 检查 output 目录是否出现 wav 与 log
4. 人工试听生成音频
5. 审核 smoke_test.log 中的推理时间和异常信息
