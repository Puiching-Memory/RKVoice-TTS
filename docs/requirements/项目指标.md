# 项目指标

## 1. 语音合成 TTS 指标

### 1.1 端侧合成延迟

- NPU 加速下，单句中文语音合成端到端延迟 ≤ 150 ms。

### 1.2 合成准确率

- 普通话合成准确率 ≥ 98%。
- 通信指令、数字、字母发音准确无歧义。
- 支持粤语、四川话等主流方言合成，方言合成准确率 ≥ 95%。

### 1.3 模型与稳定性

- INT8 量化后模型体积 ≤ 100 MB。
- 运行内存占用 ≤ 300 MB。
- 支持 7×24 小时连续稳定运行。

### 1.4 运行模式

- 纯离线端侧运行，无需网络、无需云端接口。

## 2. 语音识别 ASR 指标

### 2.1 端侧识别延迟

- 流式语音识别端到端延迟 ≤ 200 ms。

### 2.2 识别准确率

- 普通话指令识别准确率 ≥ 95%（典型噪声环境）。
- 支持主流方言语音识别，方言识别准确率 ≥ 90%。

### 2.3 模型与稳定性

- INT8 量化后模型体积 ≤ 100 MB。
- 支持流式识别。
- 支持 7×24 小时连续稳定运行。

### 2.4 运行模式

- 纯离线端侧运行，无需网络、无需云端接口。

## 3. 系统整体指标

- 全链路闭环延迟（语音→文字→语音部分）≤ 350 ms。
- 模型总存储占用 ≤ 200 MB。
- 典型工况内存占用 ≤ 500 MB。
- 支持 RK3588 NPU 硬件加速，CPU 占用可控。
- 适配短波电台音频特性。
- 可与低码率语音压缩模块无缝对接。
- 支持普通话与主流方言的混合语音交互与播报。
- 系统整体全程离线独立工作，无任何外部网络依赖。
